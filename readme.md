# Currencies

## Usage

### Require
```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/miguel-naranjatec/currencies"
    }
],
```

```
composer require  banzee/currencies --prefer-source
```

```
composer update banzee/currencies --prefer-source
```

### Search

```
dd(new Currencies)->get('eur');
```
